Dataset yang digunakan dalam penelitian ini adalah Garbage Classification Dataset.

Dataset terdiri dari 6 kategori:
- cardboard
- glass
- metal
- paper
- plastic
- trash

Dataset tidak disertakan dalam repository ini karena keterbatasan ukuran, namun dapat diakses melalui Kaggle.
https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification/data
