Deep Learning Waste Classification

Project ini bertujuan untuk mengembangkan model Convolutional Neural Network (CNN) untuk klasifikasi sampah berbasis citra.

Model dilatih menggunakan dataset Garbage Classification yang terdiri dari 6 kategori, yaitu cardboard, glass, metal, paper, plastic, dan trash.

Metode
Deep Learning (CNN)
Image preprocessing
Training menggunakan TensorFlow dan Keras
Hasil

Model mampu melakukan klasifikasi dengan tingkat akurasi yang cukup baik berdasarkan hasil training dan validasi.

Struktur Project
notebooks: berisi kode implementasi model
results: berisi hasil training dan grafik
models: berisi model yang telah dilatih
thesis: berisi laporan penelitian
